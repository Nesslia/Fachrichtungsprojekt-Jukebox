# Fachrichtungsprojekt-Jukebox
# **Zu beachten bei Benutzung des Programms!**

Für eine korrekte und fehlerlose Ausgabe in der Konsole bitte folgendes beachten:
- Konsole -> Einstellungen -> Startgröße anpassen: - Spalten 200 / Zeilen 50 - dann programm/projekt neu ausführen

## **Steuerung:**
- zum wechseln der 'Knöpfe' Pfeiltasten benutzen
- mit 'Enter' wird der 'Knopf' benutzt und das Lied wird gestartet und/oder gestoppt



UND nicht zu vergessen, Sound an! :-)
